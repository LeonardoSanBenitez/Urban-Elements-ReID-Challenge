# PAT++: a cautionary tale about generative visual augmentation for Object Re-identification

Repository for the paper [PAT++](https://arxiv.org/abs/2507.15888), by Leonardo Santiago Benitez Pereira and Arathy Jeevan, as part of the submission to the [Urban Elements ReID](https://www.kaggle.com/competitions/urban-reid-challenge) kaggle competition.

Additional media is provided in this [Google Drive](https://docs.google.com/document/d/1DBBR_yhVHE9pW_aliPRyZrROf0Kj0Cm6B0NTSSyD7Is/edit?usp=sharing).
